CappyView

CappyView is a Chrome extension that helps reviewers work smarter. It parses requisition files, streamlines case review with toolbars and sidebars, and speeds up batch searches, all directly inside the sites you already use.

Features

  • Electronic Requisition (eReq) Sidebar
    • Detects JSON and XML documents on LIMS pages.
    • Injects an in‑page sidebar into the current tab.
    • Click‑to‑find highlights and scrolls to source nodes.
    • Per‑test renderers with modular parsing.
    • Experimental customizable sidebar is available.
  • Case Toolbar
    • Quick links: Extranet, GC Form, Open All/Files/Images/Samples, Patient Search, Support Page.
  • Review and Send Toolbar
    • Quick actions: close all associated tabs
    • Quick links: Support page, Extranet
  • Support Page Toolbar
    • Navigation Bar added to allow users to scroll-to-view
    • Quick actions: Open Files/Images and save page.
  • LIMS Dashboard Table Filters
    • Filters automatically added to eligible tables upon load.
    • One-click ability to consolidate pagination.
    • Advanced filtering options added.
    • Batch case opening available.
  • Case Note Tables Enhancement
    • Note tables reformatted for cleaner view.
    • Ability to expand and collapse notes added.
  • Gallery Viewer
    • Main image area + scrollable thumbnail strip; zoom, drag zoom‑box, middle‑mouse pan.
    • Supports selecting images, keeping context of the current case ID when available.
  • Options & Customization
    • Options page added.
    • Ability to customize sidebar, case note attachments open, pages open, how pages are open (window/tab), visibility (PDF Zoom, Gallery thumbnail position, and theme / color).
  • Baylor Search
    • Sidebar to perform batch kit searches against Baylor Genetics results console.
    • CSV Export: User‑initiated export of table data; writes to local disk from the page (no remote calls).
  • Security first: no remote code, least privilege permissions, no data collection.

1. DOMPurify

Source: https://github.com/cure53/DOMPurify

License: Apache License, Version 2.0 or Mozilla Public License, Version 2.0

Copyright: © 2015–2025 Dr.-Ing. Mario Heiderich, Cure53 and other contributors

Purpose in CappyView

DOMPurify is used to sanitize JSON, XML, and HTML strings before rendering them inside the sidebar. This prevents Cross-Site Scripting (XSS) and DOM injection attacks from malicious or malformed data.

Integration Method

  • DOMPurify is bundled directly inside the extension as a static, minified .js file
  • No remote code or CDN is used
  • The version is pinned to ensure consistency and prevent unexpected updates

Security Considerations

  • DOMPurify is actively maintained and widely recognized as a best-practice library for HTML sanitization
  • It has been reviewed by security experts and is recommended in Mozilla MDN documentation for safe HTML handling in extensions
  • Updates are reviewed periodically to ensure the latest stable release is used

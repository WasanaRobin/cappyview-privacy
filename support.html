CappyView Support

Need help with CappyView? This page explains how to report issues, request enhancements, and find answers to common questions.

Important

CappyView does not collect or transmit any user or patient data. All processing occurs locally in your browser, and only non-sensitive interface preferences are stored on your device.

For full details, see the Privacy Policy.

How to Report a Problem

When requesting support, including the following details helps speed up troubleshooting:

  • What you were doing (e.g., opening the gallery, using table filters)
  • What you expected to happen
  • What actually occurred
  • Whether the issue is consistent or intermittent
  • Your Chrome version
  • The CappyView version installed

If safe to do so, you may include a screenshot with no sensitive information visible.

Basic Troubleshooting Steps

1. Confirm the extension is enabled

  • Open chrome://extensions
  • Ensure CappyView is toggled On

2. Make sure you are on a supported domain

  • CappyView only activates on authorized workplace systems.
  • If you're on an unsupported site, toolbars and sidebars will not appear.

3. Refresh the page

  • Reload (Ctrl+R / Cmd+R)
  • If still not working, close the tab and reopen your workflow entry point

4. Check for extension updates

  • Open chrome://extensions
  • Enable Developer mode (top right)
  • Click Update to force an extension refresh

5. Try a clean Chrome session

  • Close all related tabs
  • Open a new Chrome window and sign back in to your systems

Frequently Asked Questions

Does CappyView send any data outside my browser?

No. CappyView does not transmit data to external servers and performs all processing locally.

What does CappyView store?

CappyView stores only non-sensitive display preferences such as sidebar width, toolbar state, PDF zoom, and filter settings.

Why does CappyView request these permissions?

Permissions enable the extension to provide navigation and productivity features:

  • Storage – saves UI preferences
  • Scripting – enables page-aware tools
  • Tabs – manages related tabs
  • Web Navigation – syncs tools when pages change
  • Host permissions – restricts operation to authorized domains

Can I reset CappyView settings?

Yes. Reinstalling the extension or clearing Chrome extension storage resets all settings.

Who is CappyView for?

CappyView is designed for authorized personnel working within professional case-management systems. It enhances tools you already have access to without changing your permissions.

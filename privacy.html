Privacy Policy for CappyView

Last Updated: December 11, 2025

Summary: CappyView is an internal productivity tool for healthcare professionals. It does not collect, transmit, or store any patient data, personal information, or usage analytics.

What CappyView Does

CappyView is a Chrome extension that enhances internal case management workflows within Natera's LIMS (Laboratory Information Management System) and Baylor Genetics search interfaces. It adds navigation tools, customization options, and workflow improvements for authorized users.

Data Collection & Storage

CappyView does NOT collect, transmit, or store:

  • Patient health information (PHI)
  • Personal identifying information
  • Usage analytics or telemetry
  • Browsing history or activity outside specified domains
  • Any data sent to external servers

CappyView DOES store locally (on your device only):

  • User interface preferences (sidebar width, toolbar position, theme color)
  • Display settings (PDF zoom level, table filter preferences)
  • Temporary session data for image gallery performance (cleared when gallery closes)

All preference data is stored using Chrome's built-in chrome.storage.local and chrome.storage.sync APIs. This data consists solely of non-sensitive configuration values and never includes patient information or case details.

Permissions Explained

CappyView requests the following Chrome permissions. Here's why each is needed:

Storage Permission

Used to save your UI preferences (sidebar width, toolbar state, theme color, PDF zoom level) locally on your device. No patient data is ever stored.

Scripting Permission

Used only on open LIMS case tabs to run small, in-page DOM probes via chrome.scripting.executeScript. These probes discover related internal links (samples, sendout panel orders, Support Review pages, and attachment PDFs) so the "Open / Close Case Tabs" tools can correctly identify which tabs belong to the active case. No remote code is executed, no scripts are injected into non-whitelisted sites, and the results never leave the user's machine.

Tabs Permission

Used to open and manage case-related tabs (Files, Images, Samples, Support pages) based on the current case you're viewing. CappyView does not access tabs outside the specified internal domains and does not read your browsing history.

Web Navigation Permission

Detects when you navigate between pages in the single-page application to refresh toolbars and sidebars appropriately. Does not intercept, modify, or monitor network traffic.

Host Permissions

CappyView only operates on these specific internal domains:

  • https://lims.natera.com/prod-lims/app/*
  • https://lims.natera.com/extranet/app/*
  • https://generesults.baylorgenetics.com/*

These permissions allow CappyView to add functionality to these internal tools. The extension cannot access any other websites or domains.

Third-Party Code

CappyView includes the following third-party libraries:

  • DOMPurify – Used to sanitize HTML content for security purposes. This library is bundled directly in the extension (not loaded from external sources) and does not collect or transmit any data. Licensed under Apache 2.0.
  • UIcons by Flaticon – Vector icons used in the user interface. These icons are bundled in the extension and require attribution per the Flaticon Free License.

For complete license information, see the Third-Party Licenses page.

No Analytics or Tracking

CappyView includes:

  • ❌ No analytics services
  • ❌ No tracking cookies
  • ❌ No advertising
  • ❌ No telemetry or usage reporting
  • ❌ No external network requests (beyond the internal domains above)

Security

CappyView follows Chrome Manifest V3 security standards:

  • All content scripts use strict Content Security Policy
  • DOM insertion is sanitized to prevent XSS attacks
  • Message passing validates origins
  • No eval() or dynamic code execution
  • Service worker background architecture

For detailed security information, see the Security Documentation.

Your Rights

Since CappyView does not collect or store any personal data:

  • There is no personal data to access, modify, or delete
  • Local preferences can be cleared by uninstalling the extension or resetting Chrome's extension storage
  • No data is shared with third parties

Updates to This Policy

If CappyView's data practices change, this privacy policy will be updated and the "Last Updated" date will reflect the change. Continued use of the extension after updates constitutes acceptance of the new terms.
